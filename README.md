## Firebase를 사용한 이유

- ### Authentication 서비스 작성이 편리함
- ### OAuth 사용이 편리함
- ### Database를 간편하게 사용 가능

## Cloudinary를 사용한 이유

- ### firebase를 사용하면서 image를 저장하기 위한 용도로 사용

## React-Query를 사용한 이유

- ### 데이터 쿼리, 뮤테이션, 캐싱 처리가 편함

## Tailwindcss를 사용한 이유

- ### 클래스 기반 스타일링이기 때문에 스타일 작성이 빠르고 편리함
- ### 단점은 애니메이션 작성이 힘듬
